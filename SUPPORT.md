# Support

Thank you for using the **Snowdream Tech AI IDE Template**! Here's how to get help.

## 🔍 Before Asking for Help

Please check the following resources first:

1. **Documentation**: Review the [README](./README.md) and [CONVENTIONS.md](./CONVENTIONS.md).
2. **Existing Issues**: Search [open and closed issues](https://github.com/snowdreamtech/template/issues) — your question may already be answered.
3. **Discussions**: Browse [GitHub Discussions](https://github.com/snowdreamtech/template/discussions) for community Q&A.

## 💬 Getting Help

| Channel                | For What                         | Link                                                                                        |
| ---------------------- | -------------------------------- | ------------------------------------------------------------------------------------------- |
| **GitHub Discussions** | General questions, ideas, how-to | [Open a Discussion](https://github.com/snowdreamtech/template/discussions/new?category=q-a) |
| **GitHub Issues**      | Bug reports, feature requests    | [Open an Issue](https://github.com/snowdreamtech/template/issues/new/choose)                |

## 🐛 Reporting Bugs

Please use the [Bug Report template](https://github.com/snowdreamtech/template/issues/new?template=bug_report.yml).

Include:

- Your OS and version
- Steps to reproduce the issue
- Expected vs actual behavior
- Relevant logs or screenshots

## 🔒 Security Vulnerabilities

**Please do NOT open a public issue for security vulnerabilities.**

Follow the instructions in [SECURITY.md](./SECURITY.md) to report security issues privately.

## 🤝 Contributing

Interested in contributing? Read [CONTRIBUTING.md](./CONTRIBUTING.md) to get started.
